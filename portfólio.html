Correcting

Data: 2024

Correcting é um projeto criado em 2024 com o objetivo de corrigir redações de forma rápida e acessível, além de oferecer dicas e orientações para melhorar a escrita. Desenvolvido em equipe, participei da criação do design do site do zero junto com uma colega e fui responsável por programar toda a estrutura, exceto pela caixa de correção, que ficou a cargo de outra integrante. Utilizei HTML, CSS, JavaScript (para transições) e PHP para o banco de dados, além de aplicar conceitos de engenharia de prompt. Como recurso extra, integrei um chatbot usando a API do Google Gemini para ajudar os usuários na correção de textos e fornecer sugestões personalizadas.

Linguagens: HTML, CSS e JS.
